## 使用技术及框架
采用 Java 开发，数据库提供两种，默认使用 sqlite 3 ，也可以自己配置成使用 mysql。<br/>
v4.0 版本以后，使用 Spring Boot2 框架，以及采用 Maven 进行管理，更方便于开发者进行二次开发。<br/>
其他框架，包含 Shiro、Spring MVC、Hibernate等.<br/>
